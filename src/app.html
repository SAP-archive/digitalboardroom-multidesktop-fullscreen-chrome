Help

Shows a web page extended across one or more desktops without a menu bar, toolbar or title bar.

This is a Community Contribution utility and is not supported or maintained by SAP, so use this at your own risk!

Keys:

  • Help (F1)
  • Settings (F10)
  • Toggle full screen (F11)
  • Open the home page (Alt + Home)
  • Reload the current page (F5)
  • Stop the page loading (Esc)
  • New tab (Ctrl + T)
  • Next tab (Ctrl + Tab)
  • Previous tab (Ctrl + Shift + Tab)
  • Close tab (Ctrl + F4)
  • Exit (Alt + F4)

Troubleshooting

If Multi Desktop Full Screen for Chrome is not using all of your desktops:

  • Make sure the left-most desktop is set as main/primary
  • Make sure when Google Chrome is open it fits within the left-most desktop

It is best to use a desktop shortcut and then keep Google Chrome closed.
